# 🧪 3D Periodic Table — Neon Edition 🌌

Интерактивная трехмерная таблица Менделеева, выполненная в футуристичном неоновом киберпанк-стиле. Проект визуализирует химические элементы и позволяет динамически изменять форму их отображения в пространстве.

## 🌐 Живое демо (Live Demo)
👉 **[Запустить проект в браузере](https://21nizom.github.io/Neon-periodic.X)**

## 🚀 Ключевые фишки (Features)
- **5 режимов 3D-отображения:** 
  - `TABLE` — Классическая таблица элементов.
  - `SPHERE` — Элементы сворачиваются в плавно вращающийся трехмерный шар.
  - `HELIX` — Элементы выстраиваются в бесконечную спираль.
  - `GRID` — Структурированная пространственная сетка.
  - `GALAXY` — Элементы распределяются в виде космической волны/галактики.
- **Неоновая подсветка:** Каждый элемент подсвечивается кастомным цветом в зависимости от его химической группы (щелочные металлы, лантаноиды, инертные газы и т.д.).
- **Интерактивность:** Полная поддержка масштабирования (zoom) и вращения камеры мышкой или тачем на мобильном телефоне.

## 🛠️ Технологический стек (Tech Stack)
- HTML5 / CSS3 (Кастомные неоновые стили и анимации)
- JavaScript (ES6+)
- **Three.js / WebGL** (для реализации плавной 3D-графики и восстановления трансформаций)
